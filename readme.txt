== Installation ==

### Installation

1. Upload 'keek' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

### Usage

* Just use this shortcode to embedding keek video:
	[keek url="https://www.keek.com/dhoom0011/keek/D9dpeab"]

### Shortcode Attributes

1. url="here enter keek video link, default is none"
2. margin="here enter value for margin top and bottom, number only, default is 20 pixel"
3. width="here enter value for width, number only, default is 100% responsive"
4. height="here enter value for height, number only, default is 405 pixel"

### Example
	[keek url="https://www.keek.com/dhoom0011/keek/D9dpeab" width="400" height="300" margin="40"]
